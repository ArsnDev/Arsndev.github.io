CppMMO

C++ 멀티플레이어 게임 서버
개발 기간: 2개월
상태: 진행 중
C++20과 Boost.Asio를 기반으로 구현한 고성능 실시간 멀티플레이어 MMO 서버입니다. 30 TPS Fixed Timestep 게임 루프로 안정적인 실시간 처리를 제공하며, Unity 클라이언트와 함께 멀티플레이어 게임을 개발 중입니다. 클라이언트 예측(Client-side Prediction)과 서버 보정(Server Reconciliation) 시스템을 구현하여 네트워크 지연 보상 기능을 추가했습니다.
🏗️ 아키텍처 특징
  • Boost.Asio 기반 비동기 I/O
  • Server Authority 치트 방지
  • QuadTree AOI 시스템
  • FlatBuffers 직렬화
  • Docker 컨테이너 배포
⚡ 성능 최적화 사례
  • 메모리 풀링으로 동적할당 제거
  • 패킷 배칭으로 시스템콜 60% 감소
  • 비동기 로깅으로 I/O 블로킹 제거
  • AOI 연산 주기 최적화
  • TPS 조정을 통한 효율성 개선
  • Client-side Prediction 구현
  • Server Reconciliation 구현
🔧 4단계 체계적 최적화 과정
1단계: 메모리 풀링 시스템
FlatBuffers 객체와 플레이어 상태 벡터의 동적 할당 오버헤드 해결 → CPU 사용률 20% 감소
2단계: 패킷 배칭 시스템
틱 기반 배칭으로 개별 패킷 전송 최적화 → 네트워크 시스템 콜 60% 감소
3단계: 비동기 로깅
동기식 로깅의 메인 루프 블로킹 현상 해결 → I/O 대기 시간 제거
4단계: 틱레이트 최적화
TPS 30 조정 및 AOI 계산 주기 최적화 → 불필요한 연산 제거로 효율성 극대화

개발 철학 및 목표

게임 서버 개발에서 가장 중요한 것은 플레이어의 경험이라고 생각합니다. FPS 게임을 플레이하며 체감한 레이턴시의 중요성을 바탕으로, 라이엇 게임즈의 발로란트 128틱 서버와 같은 최고 수준의 게임 경험을 제공하는 서버를 구축하는 것이 제 목표입니다.
앞으로도 데이터 기반의 체계적인 최적화와 안정적인 서비스 운영을 통해 플레이어들이 몰입할 수 있는 최고의 게임 환경을 만들어가고 싶습니다.
