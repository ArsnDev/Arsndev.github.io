CppMMO

C++ 멀티플레이어 게임 서버

진행중

C++20과 Boost.Asio를 기반으로 구현한 고성능 실시간 멀티플레이어 MMO 서버입니다.

  • 30 TPS 게임 루프로 안정적인 실시간 처리
  • 개인 PC에서 600명 동시 접속 테스트 완료
  • QuadTree AOI 시스템으로 공간 최적화
  • FlatBuffers 기반 효율적인 패킷 통신
  • Server Authority 기반 치트 방지
C++20 Boost.Asio FlatBuffers Docker MySQL

OmokServer

C# 실시간 오목 게임 서버

완성

ASP.NET Core와 SignalR을 활용한 온라인 오목 게임 서버입니다. 실제로 게임을 체험해볼 수 있습니다!

  • SignalR을 통한 실시간 양방향 통신
  • JWT 기반 사용자 인증 시스템
  • Server Authority 기반 게임 로직
  • 인메모리 매치메이킹 시스템
  • 게임 상태 관리 및 승패 판정
  • 경기 결과 데이터베이스 저장
C# ASP.NET Core SignalR JWT MySQL

Chattt

C# 실시간 채팅 시스템

완성

C# Windows Forms 기반의 실시간 채팅 클라이언트/서버 시스템입니다.

  • TCP 소켓 기반 실시간 메시징
  • 사용자 인증 및 회원가입 시스템
  • 실시간 참가자 목록 관리
  • 멀티스레드 서버 아키텍처
  • Windows Forms UI
C# TCP Socket Windows Forms Threading
