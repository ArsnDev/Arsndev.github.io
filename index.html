프로젝트

CppMMO

C++ 멀티플레이어 게임 서버

완성

C++20과 Boost.Asio를 기반으로 구현한 고성능 실시간 멀티플레이어 MMO 서버입니다.

  • 60 TPS 게임 루프로 안정적인 실시간 처리
  • QuadTree AOI 시스템으로 공간 최적화
  • FlatBuffers 기반 효율적인 패킷 통신
  • Server Authority 기반 치트 방지
  • Docker 컨테이너화 배포
C++20 Boost.Asio FlatBuffers Docker MySQL

OmokServer

C# 실시간 오목 게임 서버

완성

ASP.NET Core와 SignalR을 활용한 온라인 오목 게임 서버입니다.

  • SignalR을 통한 실시간 양방향 통신
  • JWT 기반 사용자 인증 시스템
  • 인메모리 매치메이킹 시스템
  • 게임 상태 관리 및 승패 판정
  • 경기 결과 데이터베이스 저장
